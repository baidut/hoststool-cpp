hoststool-cpp

Introduction

hoststool-cpp is a C++/Qt5 port of huhamhire-hosts, provides basic tools to manage the hosts file on current operating systems. Users could use these tool to modify the hosts to visit specified websites or services blocked by ISP/government.

Features

  • huhamhire-hosts's features
  • hoststool-cpp is written in C++/Qt5, can support more platforms (IOS / Android) than huhamhire-hosts.

Note

  • The hosts list is maintained by author of huhamhire-hosts. if it's failed to access websites, please contact them.
